fsync


#include <unistd.h>


void sync(void);


int syncfs(int fd);

A função sync() executa todas as alterações de arquivos pendentes no sistema operacional


A função sync() não tem como falhar


A função syncfs() executa todas as alterações de arquivos que pendentes no sistema de arquivos em que o arquivo referido por fd está



Retorno:

int
A função syncfs() retorna 0 quando os arquivos são alterados corretamente. Caso contrário -1 é retornado, e errno é definido apropriadamente

Erros:


EBADFD fd Não é uma referência válida a um arquivo
EIO Um erro ocorreu ao tentar utilizar a mídia física que seria modificada
ENOSPC não há espaço suficiente para guardar todas as alterações
ENOSPC, EDQUOT não há espaço suficiente para guardar o arquivo (este erro apenas existe em servidores NFS)
